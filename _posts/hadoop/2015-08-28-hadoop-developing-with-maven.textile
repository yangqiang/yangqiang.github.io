# 编写程序
## 配置 pom.xml，根据节点上的hadoop版本选择对应的依赖包，务必先确认已安装的Hadoop 是 Apache 版还是 CDH 版，对于CDH版可直接参考前面资料里的配置，Apache 原生版，则从http://mvnrepository.com/找到对应版本的加入即可：搜索“hadoop-core”关键词
## 编写 Hadoop 程序，以下是一个简单的 WordCount 程序：
http://10.30.7.190:8080/yangqiang/my_java/blob/master/src/study/MRWordCount.java
# 在 idea 中使用 Maven 生成 jar 包
## 打开 Maven 操作面板：View -> Tool Windows -> Maven Projects
## 点击 Execute Maven Goal 按钮（有个M字母的按钮）
## Command Line 输入：compile
## Command Line 输入：package  参考 [[Maven]]
# 测试 Hadoop 程序
hadoop jar target-xx.jar org.xx.yy.MRWordCount input_dir output_dir
